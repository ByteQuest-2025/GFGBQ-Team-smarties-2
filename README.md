## Project Overview

- TruthCheck AI verifies AI-generated text for factual correctness
- Extracts individual claims from user input
- Classifies content as Verified, False, or Uncertain using trusted knowledge sources

## Setup & Installation

- Clone the project repository from GitHub
- Install required Python dependencies
- Run the application using Streamlit

## How It Works (Usage)

- User pastes AI-generated text into the interface
- System extracts factual statements automatically
- Each statement is verified and shown with:
 - Green → Verified
 - Red → False
